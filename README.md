# README: Installing Homebrew and Poetry on macOS

This guide will walk you through installing **Homebrew** (a package manager for macOS) and **Poetry** (a dependency manager for Python) on your Mac machine.

---

## **Prerequisites**

1. Ensure you have macOS installed and updated.

---

## **Step 1: Installing Homebrew**

**Homebrew** simplifies software installation on macOS. Follow these steps to install Homebrew:

1. **Open Terminal**:  
   Press `Command + Space`, type `Terminal`, and press `Enter`.

2. **Run the Homebrew installation script**:  
   Paste the following command in the terminal and press `Enter`:
